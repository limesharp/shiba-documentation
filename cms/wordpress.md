# CMS Pages - WordPress

-   CMS Pages are created in WordPress using WPAlchemy MetaBoxes
-   Each CMS page has a WordPress template
-   Please find information about WPAlchemy MetaBoxes [here](https://www.farinspace.com/wpalchemy-metabox/)
-   After creating the WordPress templates add some test content by creating a page in WordPress. Don't forget to select the page template you just created.
-   Once the content is added you can see it in the front end after implementing the Vue part
